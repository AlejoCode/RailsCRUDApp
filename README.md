## Getting Ruby Set up and necessary gems
* Install the gems listed in the Gemfile by :
```bash
$ bundle install
```
* Migrate the database by:
```bash
$ rails db:create and then, $ rails db:migrate
```
* Simply run the app with :
```bash
$ rails s
```
